КОНТЕКСТ: В этом репозитории есть две папки: корневая (Node.js - активная) и php/ (неактивная, только для истории).

Правила для LLM:
1. Игнорируйте все файлы в папке php/ при анализе работающего кода
2. Предложения по улучшению делайте только для Node.js части
3. PHP-код используйте ТОЛЬКО для понимания бизнес-логики (алгоритмы расчетов)
4. Не предлагайте исправления для PHP, не анализируйте PHP-зависимости

Актуальная версия: bot.js, main.js, scheduler.js, database.js
Неактуальная версия: php/ (любые файлы внутри)
